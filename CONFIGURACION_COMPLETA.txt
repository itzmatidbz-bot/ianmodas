🔧 PASOS PARA ACTIVAR (IMPORTANTE):

1. CONFIGURAR SUPABASE EDGE FUNCTIONS:
   📁 Crear carpetas en tu proyecto:
   
   supabase/
   └── functions/
       ├── send-welcome-email/
       │   └── index.ts (copiar contenido de edge-function-send-welcome-email.ts)
       └── generate-product-description/
           └── index.ts (copiar contenido de edge-function-generate-descriptions.ts)

2. OBTENER API KEYS:
   
   📧 RESEND (para emails):
   - Ir a: https://resend.com
   - Crear cuenta gratis
   - Obtener API Key
   
   🤖 OPENAI (para ChatGPT):
   - Ir a: https://platform.openai.com
   - Crear cuenta
   - Obtener API Key (necesitas agregar $5-10 de crédito)

3. CONFIGURAR VARIABLES EN SUPABASE:
   - Ir a Supabase Dashboard → Settings → Edge Functions
   - Agregar variables de entorno:
     * RESEND_API_KEY: tu_key_de_resend
     * OPENAI_API_KEY: tu_key_de_openai

4. DESPLEGAR FUNCIONES:
   ```bash
   supabase functions deploy send-welcome-email
   supabase functions deploy generate-product-description
   ```

5. EJECUTAR SCRIPTS SQL:
   - Ejecutar RESET_COMPLETO.sql
   - Ejecutar DETECTAR_USUARIOS_REALES.sql
